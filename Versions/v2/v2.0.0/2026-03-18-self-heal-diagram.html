A chatbot that teaches itself. Every unanswered question

AGENTIC WORKFLOW
Before — Static Bot
1 User asks about sweep picking
2 Bot: "I don't have that information"
3 Developer manually writes data file
4 Developer generates embeddings, deploys
5 Bot can finally answer
Time to learn: days to weeks
After — Self-Healing Bot
1 User asks about sweep picking
2 Bot responds with what it can
3 Agent generates knowledge in background
4 Agent validates, embeds, and stores
5 Bot can answer on the next question
Time to learn: ~16 seconds
How it learns
User asks a question
RAG Retrieval + Confidence Score
Embed query, search knowledge base, score relevance
Confident — score >= 0.5
Answer from knowledge
Bot knows this topic. Respond normally.
Knowledge Gap — score < 0.5
Respond + trigger agent
Immediate reply. Background agent starts learning.
Background Agent
Non-blocking — user never waits
1 Filter — skip greetings, gibberish can halt
2 Boundary check — LLM: is this in-domain? can halt
3 Duplicate check — embedding similarity > 0.7? can halt
4 Generate — LLM writes structured knowledge entry
5 Validate — second LLM fact-checks for accuracy can halt
6 Learn — upload to S3 + embed + store in DynamoDB
7 Notify — bust cache + email + queue user alert
Knowledge base grows — bot is now smarter
